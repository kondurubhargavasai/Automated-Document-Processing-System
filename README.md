This project is about Automated Document Processing, Categorization and Report Generation using Python. In this project, I created a system that can read different types of documents like PDF, Word (.docx), Text (.txt), and Excel (.xlsx) files. All the input files are placed inside the input_documents folder. The program automatically reads each file and extracts the text from it. After extracting the text, it checks some predefined keywords and categorizes the document into HR, Finance, Operations, Legal, or Unknown. This categorization is done using simple keyword matching logic and not using machine learning. Then for every document, the system generates a short summary of around 4–5 lines, extracts important keywords, finds dates and currency amounts using regular expressions, and also calculates word count and character count. After that, it generates a detailed PDF report using the reportlab library and saves it inside the output/reports folder. It also saves summary files and category files in separate folders inside the output directory. I also implemented a logging system which records the document name, category, timestamp, and errors (if any) in a system.log file inside the logs folder. The project is divided into different files like main.py, file_reader.py, classifier.py, report_generator.py, and logger.py to make the code clean and easy to understand. main.py controls the full execution flow. Through this project, I learned about file handling, modular programming, regular expressions, PDF generation, and how automation systems work in real applications. In future, this project can be improved by adding machine learning for better classification or by creating a simple web interface.
